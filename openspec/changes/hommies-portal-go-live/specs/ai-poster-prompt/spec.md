## ADDED Requirements

### Requirement: Gemini generates the poster prompt server-side

The "Generate poster prompt" card on the Add Property screen SHALL produce the `/room-showcase-pdf` brief by calling a Convex action that runs Gemini, passing the property's details, and returning the generated prompt text to the UI.

#### Scenario: Admin generates a tailored prompt

- **WHEN** the admin triggers prompt generation with the property basics filled in
- **THEN** the portal calls the Convex action, Gemini writes a `/room-showcase-pdf` brief from that property's details, and the prompt is shown for the admin to copy

#### Scenario: Generation requires the property basics

- **WHEN** the admin triggers prompt generation before condo, area, and rent are filled in
- **THEN** the portal does not call Gemini and prompts the admin to fill the basics first

### Requirement: Gemini API key stays server-side

The Gemini API key SHALL be held only as a Convex environment variable and SHALL never be bundled into or exposed by the client.

#### Scenario: Key is not in the client bundle

- **WHEN** the built client bundle is inspected
- **THEN** the Gemini API key does not appear in it; only the Convex action references the key

### Requirement: Static template fallback when Gemini is unavailable

When the Gemini call fails, times out, or the API key is unset, the action SHALL return the deterministic static-template prompt instead, and the UI SHALL indicate which path produced the prompt.

#### Scenario: Gemini outage falls back to the template

- **WHEN** the Gemini call errors or times out
- **THEN** the action returns the static-template `/room-showcase-pdf` brief and the workflow is not blocked

#### Scenario: Prompt source is surfaced

- **WHEN** a prompt is returned to the UI
- **THEN** the UI indicates whether it was written by Gemini or produced by the template fallback

### Requirement: Brand-constrained, correctly structured brief

The generated prompt SHALL begin with `/room-showcase-pdf`, embed the Hommies brand rules (orange `#fd6925`, navy `#041f60`, cream `#fff5ec`, family-first tone, the mandatory "we are not agents" footer line), and instruct an A4 single-page poster that makes the four facts unmissable — room type, location & area, Condo or HDB, and age of building.

#### Scenario: Generated brief carries the brand guardrails

- **WHEN** a prompt is generated by Gemini
- **THEN** it starts with `/room-showcase-pdf`, names the brand colours and tone, requires the disclaimer footer, and calls for an A4 single page surfacing the four mandatory facts

#### Scenario: Generated brief reflects the property

- **WHEN** a prompt is generated for a specific property
- **THEN** it includes that property's name, building type, area, age, room type, rent, commute numbers, and recorded media
